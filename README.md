# Sales Analytics Platform

## Overview
Developed an end-to-end analytics solution in Microsoft Fabric to ingest, transform, model, and visualize sales data for business reporting and decision-making.

## Tech Stack
- Microsoft Fabric
- OneLake
- Lakehouse
- Dataflow Gen2
- Warehouse
- SQL
- Power BI

## Data Sources
- Sales Transactions
- Customer Master
- Product Master
- Region Master

## Solution Flow
Source Data → Lakehouse → Dataflow Gen2 → Warehouse → Semantic Model → Power BI Dashboard

## Key Metrics
- Revenue
- Order Count
- Average Order Value
- Gross Margin
- Sales by Region
- Product Performance
- Customer Performance
